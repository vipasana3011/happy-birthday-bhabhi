# Surbhi — A Cinematic Birthday Experience 💖

Ultra-premium animated birthday site built with **Vite + React + TypeScript + Tailwind + Framer Motion**.
100% static — deploys cleanly to Netlify.

## Run locally
```bash
npm install
npm run dev
```

## Build for production
```bash
npm install
npm run build
```
The output goes to `/dist`. Upload that folder to Netlify (or drag & drop into Netlify's deploy UI).

## Custom assets
Place your files inside `public/assets/`:
- `TumHoToh.mp3` — background song
- `pic1.jpg` … `pic6.jpg` — polaroid photos
- (optional) `gift1.gif` … etc.
